# To-Do Project

Данный проект направлен на разработку веб-приложения для управления личными задачами (To-Do List). 
Приложение позволит пользователям создавать, редактировать и отслеживать выполнение задач.

Проект выполняется в рамках лабораторной работы №10 по дисциплине, с использованием GitHub Projects для управления задачами команды.
